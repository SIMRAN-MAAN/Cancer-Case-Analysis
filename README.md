# Cancer Case Dashboard – Power BI

This project presents an interactive Cancer Case Dashboard developed using Power BI to analyze and visualize cancer-related data. The dashboard provides insights into patient demographics, cancer types, diagnosis trends, treatment outcomes, and survival statistics through dynamic visualizations and KPIs.

## Key Features

* Interactive filters and slicers for detailed data exploration
* Analysis of cancer cases by age, gender, and region
* Visualization of cancer type distribution and trends
* Treatment outcome and survival rate analysis
* Key performance indicators (KPIs) for quick insights
* User-friendly and responsive dashboard design

## Tools & Technologies

* Power BI Desktop
* DAX (Data Analysis Expressions)
* Power Query
* Data Modeling
* Data Visualization

## Project Objectives

The main objective of this dashboard is to transform raw healthcare data into meaningful insights that can support data-driven decision-making, identify trends, and improve understanding of cancer case patterns.

## Insights Delivered

* Identification of high-risk demographic groups
* Most common cancer types and their prevalence
* Trends in diagnoses over time
* Treatment effectiveness and patient outcomes
* Regional distribution of cancer cases

This dashboard demonstrates the use of business intelligence techniques to analyze healthcare data and communicate insights effectively through interactive reporting.
